# RehireEmploymentStatus

The employee's employment status. Supplying an invalid option will set the employment_status to *not_set*.

| ------------------ | ------------------ |
| `PartTime`         | part_time          |
| `FullTime`         | full_time          |
| `PartTimeEligible` | part_time_eligible |
| `Variable`         | variable           |
| `Seasonal`         | seasonal           |
| `NotSet`           | not_set            |
