# PaymentPeriod

How often the agency collects the withholding amount. e.g. $500 monthly -> `Monthly`.


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `EveryWeek`      | Every week       |
| `EveryOtherWeek` | Every other week |
| `TwicePerMonth`  | Twice per month  |
| `Monthly`        | Monthly          |
